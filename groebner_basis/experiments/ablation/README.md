# Gröbner basis — ablation experiment

**Goal**: study how training monomial order affects exact-match accuracy.

| Knob | Choices |
|------|---------|
| `--training_order` | `degrevlex` (use stored basis as-is), `lex` (FGLM at load time) |

Lex bases are typically larger and have larger coefficients, so accuracy is
expected to drop vs degrevlex (see Table 5 of the ISSAC2026 paper: 42.4% degrevlex
vs 32.4% lex on QQ).

**Implementation note**: the lex Gröbner basis is computed at load time with
`libsingular:std` explicitly (`core/parser.py`). The default `stdhilb` algorithm
in SageMath 10.6 raises an `intvec` / `bigintvec` error on some QQ-lex inputs;
forcing `libsingular:std` is the standard workaround.

## How to run
```bash
cd groebner_basis/experiments/ablation/scripts
python train.py --training_order degrevlex
python train.py --training_order lex
```

Reuses the dataset generated by the toy experiment (`../../toy/data/QQ/`).
Save dirs are auto-suffixed with `_degrevlex` / `_lex` to avoid clobbering.
