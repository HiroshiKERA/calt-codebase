# Border basis — ablation experiment

**Goal**: vary the model size and observe how performance changes on the toy
border-basis dataset (GF(7), 2 variables).

| Knob | Choices |
|------|---------|
| `--model_size` | `small`, `medium`, `large` |

## How to run
```bash
cd border_basis/experiments/ablation/scripts
python train.py --model_size large
```

Reuses the dataset generated by the toy experiment (`../../toy/data/GF7/`).
Outputs land in `outputs/results_<model_size>/`.
